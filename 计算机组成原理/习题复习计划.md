## ✅ 复习检查清单

### 易错点自查
- [ ] 1KB = $2^{10}$ Byte（不是 bit）
- [ ] 补码 -128 = `1000 0000`
- [ ] Cache 行数 vs 组数的区别
- [ ] 直接映射 vs 组相联公式

### 高频公式回顾
```
寻址范围 = 2^地址位数
Cache组数 = Cache行数 ÷ 路数(Ways)
RAID1存储密度 = 50%
RAID5存储密度 = (N-1)/N
```
