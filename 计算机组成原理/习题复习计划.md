## 📌 重点章节速记

### Ch4 Cache 核心公式

- **直接映射**：Cache行号 = 主存块号 mod Cache总行数
- **组相联映射**：组号 = 主存块号 mod 组数
- **块号**：地址 ÷ 块大小

### Ch6 磁盘访问时间

- **平均访问时间** = 平均寻道时间 + 平均旋转延迟 + 传输时间
- **平均旋转延迟** = 1/(2×转速)
- **传输速率** = 转速 × 每磁道数据量

### Ch10 补码计算

- **正数补码** = 原码
- **负数补码** = 绝对值取反 + 1
- **溢出判断**：两同号数相加结果符号相反则溢出

### Ch10 IEEE 754

- **单精度**：1位符号 + 8位阶码 + 23位尾数
- **阶码偏移**：127（单精度）/ 1023（双精度）
- **规格化**：1.xxxx × 2^E

### Ch14 指令周期

```
Fetch → [Indirect] → Execute → [Interrupt]
```

### Ch19 典型微操作

- **取指**：PC→MAR, M(MAR)→IR, PC+1→PC
- **间址**：IR(AD)→MAR, M(MAR)→MBR
