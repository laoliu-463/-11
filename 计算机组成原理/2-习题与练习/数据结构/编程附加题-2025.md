## 二、填空题（共1题，20分）

### 21. 单链表删除值为x的元素

```c
typedef int DataType;     /*定义DataType为整型*/

typedef struct Node {
    DataType data;
    struct Node *next;
} LinkList;

void deleteX(LinkList *L, DataType x) {
    LinkList *p, *q;
    p = L;                              /*p从头结点开始*/
    
    while(p->next && p->next->data != x)    /*第1空: p->next->data != x*/
    {
        p = p->next;                    /*第2空: p = p->next*/
    }
    
    if(!p->next) return;               /*空表直接退出*/
    
    q = p->next;
    p->next = q->next;                 /*第3空: p->next = q->next*/
    free(q);                           /*第4空: free(q)*/
}
```

**答案**:
| 空号 | 答案 |
|:---|:---|
| 第1空 | `int` |
| 第2空 | `p->next->data != x` |
| 第3空 | `p = p->next` |
| 第4空 | `p->next = q->next` |
| 第5空 | `free(q)` |
