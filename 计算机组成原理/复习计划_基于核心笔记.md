# 计算机组成原理复习计划 (基于核心考点笔记)

> **说明**：本计划严格依据 `计算机组成原理-核心考点笔记.md` 制定，涵盖所有已整理的考点与习题类型。

## 📅 复习阶段规划

### 第一阶段：核心概念扫盲 (Core Concepts)
**目标**：掌握基础定义与原理，解决选择题中的概念辨析。

#### 1. I/O 系统 (Module 4)
*   **控制方式对比**：
    *   **程序查询**：CPU 忙等。
    *   **中断驱动**：CPU 并行，指令结束时响应。
    *   **DMA**：硬件接管总线，周期窃取，仅开始/结束中断 CPU。
*   **总线与接口**：
    *   总线分类 (数据/地址/控制)。
    *   同一时刻只能有一个主设备控制总线。

#### 2. 存储系统 (Module 2)
*   **层次结构**：Reg > Cache > Mem > Disk。
*   **介质特性**：
    *   **SRAM** (6T, 快, Cache) vs **DRAM** (1T1C, 慢, 主存, 需刷新)。
    *   **ROM** (非易失)。
*   **RAID 级别**：
    *   **RAID 0** (快), **RAID 1** (镜像), **RAID 5** (分布式校验)。
    *   **独立访问** (RAID 4/5/6) vs **并行访问** (RAID 2/3)。

#### 3. 指令与 CPU (Module 3)
*   **体系结构**：冯·诺依曼 (存储程序)。
*   **RISC vs CISC**：RISC (指令简, 寄存器多, Load/Store)。
*   **流水线**：
    *   性能公式 $T = (k+n-1)\Delta t$。
    *   **冒险**：结构、数据、控制 (条件分支最严重)。

### 第二阶段：计算题攻坚 (Calculation Mastery)
**目标**：拿下必考大题与复杂计算选择题。

#### 1. 数据表示 (Module 1)
*   **补码运算**：
    *   **范围**：$-2^{n-1} \sim 2^{n-1}-1$。
    *   **溢出判断**：正+正=负，负+负=正。
    *   **最小值陷阱**：8位补码最小是 `-128` (`10000000`)，但题目常考 `-127` (`10000001`)。
*   **海明码 (Hamming Code)**：
    *   **校验位**：$1, 2, 4, 8$。
    *   **故障字**：$P_8 P_4 P_2 P_1$ 定位错误位。

#### 2. 存储器计算 (Module 2)
*   **Cache 映射 (重中之重)**：
    *   **地址拆分**：Tag | Set | Offset。
    *   **组索引计算**：$Addr / BlockSize \pmod{Sets}$。
    *   **容量扩展**：位扩展 (并联) vs 字扩展 (串联+片选)。
*   **地址线计算**：容量/位宽 = 单元数 => 地址线根数。

#### 3. 指令系统 (Module 3)
*   **操作码扩展**：$(N_{total} - N_{used}) \times 2^k$。
*   **寻址方式有效地址 (EA)**：
    *   **相对寻址**：$PC + Offset$ (注意 PC 预取后已+1)。
    *   **变址寻址**：$IX + A$ (数组)。

#### 4. 中断掩码 (Module 4)
*   **掩码字**：屏蔽自身 + 低优先级。
*   **时间轴**：抢占 (嵌套) vs 等待。

### 第三阶段：全真模拟与查漏补缺 (Final Sprint)
**目标**：通过综合习题巩固记忆。

#### 1. 综合练习 (104 & 111/113)
*   **MESI 协议**：写失效 (Write-Invalidate)，状态转换 (M/E/S/I)。
*   **中断全流程**：请求 -> 响应 -> 保存(PC/PSW) -> ISR -> 恢复 -> 返回。
*   **微操作**：理解取指周期 (Fetch) 的 MBR/MAR 数据流。
