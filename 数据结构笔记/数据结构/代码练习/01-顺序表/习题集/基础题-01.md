# 基础题-01：顺序表核心操作实现

## 题目
实现顺序表（Sequence List）的初始化、插入、删除、查找与遍历五个基础操作，要求：
1. 使用定长顺序存储结构（数组 + 当前长度）；
2. 插入位置合法且元素后移，删除位置合法且元素前移；
3. 查找返回元素首次出现的位置（1 基）；
4. 提供遍历函数输出所有元素。

## 输入输出
- **输入**：顺序表最大容量 `MaxSize`，初始元素个数 `n` 及元素列表；随后输入一条插入操作（位置、元素）、一条删除操作（位置）和一个待查找的元素值。
- **输出**：
  1. 插入后的顺序表序列；
  2. 删除后的顺序表序列；
  3. 查找结果（位置或 -1 表示不存在）。

## 样例
输入：
```
MaxSize = 10
n = 5
初始序列：2 4 6 8 10
插入：pos = 3, elem = 5
删除：pos = 5
查找：elem = 8
```

输出：
```
After Insert: 2 4 5 6 8 10
After Delete: 2 4 5 6 8
Locate(8) = 5
```

## 思路提示
1. 顺序表结构包含 `data[MaxSize]` 与 `length`；
2. 插入：检查满表与区间 `1 ≤ pos ≤ length+1`，从尾到 pos 依次右移；
3. 删除：检查空表与区间 `1 ≤ pos ≤ length`，从 pos+1 到尾部依次左移；
4. 查找：线性扫描并返回位置；
5. 遍历：按 `0 … length-1` 输出。

## 时间与空间复杂度
- 插入/删除/查找的最坏时间复杂度均为 `O(n)`；
- 额外空间复杂度 `O(1)`（除顺序表本身）。

## 参考答案（要点）
```pseudo
InitList(L): L.length ← 0
Insert(L, pos, x):
    if L.length == MaxSize or pos < 1 or pos > L.length+1: return false
    for i ← L.length-1 downto pos-1:
        L.data[i+1] ← L.data[i]
    L.data[pos-1] ← x
    L.length++
Delete(L, pos):
    if L.length == 0 or pos < 1 or pos > L.length: return false
    for i ← pos to L.length-1:
        L.data[i-1] ← L.data[i]
    L.length--
Locate(L, x):
    for i ← 0 to L.length-1:
        if L.data[i] == x: return i+1
    return -1
Traverse(L): 输出 L.data[0..length-1]
```
