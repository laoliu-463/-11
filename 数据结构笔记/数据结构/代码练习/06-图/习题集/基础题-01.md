# 基础题-01：图的邻接表表示

## 题目

**问题**：实现图的邻接表表示法

**要求**：
1. 定义图的邻接表数据结构
2. 实现图的创建函数
3. 实现添加边的函数
4. 实现打印图的函数

---

## 输入输出

- **输入**：图的顶点数和边数，以及每条边的信息
- **输出**：打印邻接表表示的图

---

## 样例

**输入**：
```
顶点数：5
边数：6
边信息：
0 1
0 2
1 3
2 3
2 4
3 4
```

**输出**：
```
顶点0: 1 -> 2 -> NULL
顶点1: 0 -> 3 -> NULL
顶点2: 0 -> 3 -> 4 -> NULL
顶点3: 1 -> 2 -> 4 -> NULL
顶点4: 2 -> 3 -> NULL
```

---

## 思路提示

1. **数据结构设计**：
   - 使用链表数组存储每个顶点的邻接顶点
   - 每个顶点对应一个链表

2. **创建图**：
   - 初始化顶点数组
   - 为每个顶点创建空链表

3. **添加边**：
   - 在对应顶点的链表中添加邻接顶点
   - 如果是无向图，需要添加两条边

4. **打印图**：
   - 遍历每个顶点
   - 打印该顶点的邻接链表

---

## 时间与空间复杂度

- **时间复杂度**：
  - 创建图：O(V)
  - 添加边：O(1)
  - 打印图：O(V + E)

- **空间复杂度**：O(V + E)

---

## 参考答案（可选）

```java
// 邻接表节点
class AdjListNode {
    int vertex;
    AdjListNode next;
    
    AdjListNode(int v) {
        vertex = v;
        next = null;
    }
}

// 图类
class Graph {
    int V;  // 顶点数
    AdjListNode[] adj;  // 邻接表数组
    
    Graph(int v) {
        V = v;
        adj = new AdjListNode[V];
        for (int i = 0; i < V; i++) {
            adj[i] = null;
        }
    }
    
    // 添加边（无向图）
    void addEdge(int src, int dest) {
        // 添加src到dest的边
        AdjListNode newNode = new AdjListNode(dest);
        newNode.next = adj[src];
        adj[src] = newNode;
        
        // 添加dest到src的边
        newNode = new AdjListNode(src);
        newNode.next = adj[dest];
        adj[dest] = newNode;
    }
    
    // 打印图
    void printGraph() {
        for (int i = 0; i < V; i++) {
            System.out.print("顶点" + i + ": ");
            AdjListNode current = adj[i];
            while (current != null) {
                System.out.print(current.vertex);
                if (current.next != null) {
                    System.out.print(" -> ");
                }
                current = current.next;
            }
            System.out.println(" -> NULL");
        }
    }
}
```
