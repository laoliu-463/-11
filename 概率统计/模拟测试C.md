## 三、解答题（12分）

设随机变量 $X$ 的概率密度为：
$$f(x) = \begin{cases} Ae^{-x}, & x > 0 \\ 0, & x \le 0 \end{cases}$$
求：
**(1)** 常数 $A$；
**(2)** $X$ 的分布函数 $F(x)$；
**(3)** $P\{-1 < X < 0.5\}$。

## 四、解答题（10分）

二维连续型随机变量 $(X, Y)$ 的联合概率密度函数为：
$$f(x, y) = \begin{cases} kx^2y, & 0 < x < y < 1 \\ 0, & \text{其他} \end{cases}$$
求：
**(1)** 常数 $k$；
**(2)** 边缘概率密度 $f_X(x), f_Y(y)$，并判断 $X, Y$ 的独立性；
**(3)** $E(X)$。

## 五、解答题（10分）

一个箱子有100件产品，其中一、二、三等品分别为80, 10, 10件，现从中随机抽取一件，令：
$$X_i = \begin{cases} 1, & \text{抽到 } i \text{ 等品} \\ 0, & \text{其他} \end{cases}, \quad i=1,2,3$$
求：
**(1)** $(X_1, X_2)$ 的联合分布律；
**(2)** $\rho_{X_1, X_2}$，并判别 $X_1$ 与 $X_2$ 是否互不相关。
