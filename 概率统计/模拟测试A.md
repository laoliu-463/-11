## 二、选择题（每题3分，共15分）

**1.** 若随机变量 $X$ 在 $(1, 6)$ 上服从均匀分布，则方程 $x^2 + Xx + 1 = 0$ 有实根的概率为 ( )
Ⓐ 0.6  Ⓑ 0.8  Ⓒ 0.7  Ⓓ 0.5

**2.** 对于任意事件 $A, B$，下列说法正确的是 ( )
Ⓐ $AB \neq \emptyset$, 则 $A$ 与 $B$ 一定独立
Ⓑ $AB \neq \emptyset$, 则 $A$ 与 $B$ 有可能独立
Ⓒ $AB = \emptyset$, 则 $A$ 与 $B$ 一定独立
Ⓓ $AB = \emptyset$, 则 $A$ 与 $B$ 一定不独立

**3.** 对任意随机变量 $X$，若 $E(X)$ 存在，则 $E(D(E(X)))$ 等于 ( )
Ⓐ 0  Ⓑ $X$  Ⓒ $(E(X))^3$  Ⓓ $E(X)$

**4.** 设 $X \sim N(1, 3^2)$，$X_1, X_2, \dots, X_n$ 为 $X$ 的样本，则下列正确的是 ( )
Ⓐ $(\bar{X} - 1) / 3 \sim N(0, 1)$
Ⓑ $(\bar{X} - 1) / 9 \sim N(0, 1)$
Ⓒ $(\bar{X} - 1) / (3 / \sqrt{n}) \sim N(0, 1)$
Ⓓ $(\bar{X} - 1) / \sqrt{3} \sim N(0, 1)$

**5.** 已知 $X \sim t(n), (n > 1)$，且 $Y = X^2$，则 ( )
Ⓐ $Y \sim \chi^2(n)$  Ⓑ $Y \sim \chi^2(n-1)$  Ⓒ $Y \sim F(n, 1)$  Ⓓ $Y \sim F(1, n)$

**(1)** 当发生意外时，两个系统至少有一个有效工作的概率；
**(2)** 在系统 $b$ 失灵的情况下，系统 $a$ 有效的概率。
