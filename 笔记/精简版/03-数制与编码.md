## 一、数制转换

### 什么是数制？

**通俗理解：** 数制就像"不同的计数方式"，十进制用0-9，二进制用0-1，十六进制用0-9和A-F。

```
常用数制：
  • 二进制（Binary）：0, 1
  • 八进制（Octal）：0-7
  • 十进制（Decimal）：0-9
  • 十六进制（Hexadecimal）：0-9, A-F
```

### 进制转换

**二进制 ↔ 十进制**

```
二进制 → 十进制：
  1011₂ = 1×2³ + 0×2² + 1×2¹ + 1×2⁰
       = 8 + 0 + 2 + 1
       = 11₁₀

十进制 → 二进制：
  11 ÷ 2 = 5 余 1
   5 ÷ 2 = 2 余 1
   2 ÷ 2 = 1 余 0
   1 ÷ 2 = 0 余 1
  
  结果：1011₂（从下往上读）
```

**二进制 ↔ 十六进制**

```
二进制 → 十六进制：
  每4位一组，转换为十六进制
  1011 0110₂ = B6₁₆
  
  1011 = B
  0110 = 6

十六进制 → 二进制：
  每位转换为4位二进制
  B6₁₆ = 1011 0110₂
```

## 二、BCD码

### 什么是BCD码？

**定义：** BCD码（Binary Coded Decimal，二进制编码的十进制）是用4位二进制表示一位十进制数的编码方式。

**通俗理解：** BCD码就像"用二进制表示十进制"，每个十进制位用4位二进制表示。

```
BCD码规则：
  • 用4位二进制表示0-9
  • 每4位表示一个十进制位
  • 1010-1111不使用（非法）

示例：
  十进制：123
  BCD码：0001 0010 0011
  
  1 → 0001
  2 → 0010
  3 → 0011
```

### BCD码特点

```
优点：
  ✓ 转换简单（直接对应）
  ✓ 精度高（无舍入误差）

缺点：
  ✗ 存储效率低（浪费空间）
  ✗ 运算复杂（需要调整）

应用：
  • 金融计算（需要精确）
  • 显示系统（直接显示）
```

## 三、ASCII码

### 什么是ASCII码？

**定义：** ASCII码（American Standard Code for Information Interchange）是用7位二进制（0-127）表示字符的编码标准。

**通俗理解：** ASCII码就像"字符的编号"，每个字符对应一个数字。

```
ASCII码范围：
  • 0-31：控制字符
  • 32-126：可打印字符
  • 127：删除字符

常用字符：
  '0' = 48₁₀ = 30₁₆
  'A' = 65₁₀ = 41₁₆
  'a' = 97₁₀ = 61₁₆
  ' ' = 32₁₀ = 20₁₆（空格）
```

### ASCII码表（部分）

```
可打印字符：
  0-9: 48-57 (30₁₆-39₁₆)
  A-Z: 65-90 (41₁₆-5A₁₆)
  a-z: 97-122 (61₁₆-7A₁₆)

控制字符：
  LF (换行): 10 (0A₁₆)
  CR (回车): 13 (0D₁₆)
  ESC: 27 (1B₁₆)
```

## 四、字符编码

### 字符编码发展

**通俗理解：** 字符编码就像"字符的编号系统"，不同语言需要不同的编码。

```
字符编码发展：
  1. ASCII（7位，128个字符）
     • 只能表示英文字符
     • 无法表示中文等其他语言
  
  2. 扩展ASCII（8位，256个字符）
     • 增加128个字符
     • 仍无法表示中文
  
  3. Unicode（统一编码）
     • 统一所有字符的编码
     • UTF-8、UTF-16、UTF-32
     • 可以表示所有语言
```

### UTF-8编码

```
UTF-8特点：
  • 变长编码（1-4字节）
  • 兼容ASCII（ASCII字符用1字节）
  • 广泛使用

编码规则：
  • ASCII字符（0-127）：1字节
  • 其他字符：2-4字节

示例：
  'A' = 41₁₆（1字节，与ASCII相同）
  '中' = E4 B8 AD（3字节）
```

## 五、总结

### 核心要点

```
✅ 数制转换：二进制、八进制、十进制、十六进制
✅ BCD码：用4位二进制表示一位十进制
✅ ASCII码：7位编码，表示128个字符
✅ 字符编码：ASCII → 扩展ASCII → Unicode
✅ UTF-8：变长编码，兼容ASCII，广泛使用
```

### 关键理解

- **数制转换**：不同进制之间的转换方法
- **BCD码**：用二进制表示十进制，精度高但效率低
- **ASCII码**：字符的编号，0-127表示128个字符
- **字符编码**：从ASCII到Unicode的发展
- **UTF-8**：变长编码，兼容ASCII，广泛使用

### 应用场景

```
应用场景：
  • 数制转换：理解计算机内部表示
  • BCD码：金融计算、显示系统
  • ASCII码：文本处理、字符比较
  • UTF-8：现代文本编码标准
```
