# 01-计算机体系结构概述 · 习题集

---

## 📚 学习目标

通过本习题集，你应该能够：
- 理解计算机体系结构的基本概念
- 掌握冯诺依曼结构的特点
- 了解计算机发展历史
- 区分体系结构与组成原理

---

## 📝 习题列表

### 基础题

1. **基础题-01：体系结构定义**
   - 题目：什么是计算机体系结构？它与组成原理的区别是什么？
   - 类型：简答题
   - 难度：⭐⭐

2. **基础题-02：冯诺依曼结构特点**
   - 题目：冯诺依曼结构的五大组成部分是什么？
   - 类型：填空题
   - 难度：⭐⭐

3. **基础题-03：体系结构层次**
   - 题目：体系结构分为哪三个层次？各层次关注什么？
   - 类型：简答题
   - 难度：⭐⭐

### 进阶题

1. **进阶题-01：体系结构vs组成原理**
   - 题目：举例说明体系结构和组成原理的区别
   - 类型：分析题
   - 难度：⭐⭐⭐

2. **进阶题-02：冯诺依曼结构分析**
   - 题目：分析冯诺依曼结构的优缺点，并说明现代计算机的改进
   - 类型：分析题
   - 难度：⭐⭐⭐

### 综合题

1. **综合题-01：计算机发展历史**
   - 题目：梳理计算机发展的各个阶段，分析每个阶段的特点和代表性技术
   - 类型：综合题
   - 难度：⭐⭐⭐⭐
