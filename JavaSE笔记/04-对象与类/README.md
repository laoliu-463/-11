# 对象与类（大白话）

类是“图纸”，对象是“成品”。写 Java 就是在写类，然后造对象。

---

## 1. 类与对象

- 类：属性 + 方法的集合
- 对象：类的实例

---

## 2. 构造器（new 时执行）

```java
class Person {
    String name;
    Person(String name) { this.name = name; }
}
```

构造器名和类名相同，没有返回值。

---

## 3. this 关键字

`this` 代表当前对象，用来区分“成员变量 vs 参数”。

---

## 4. static（类级别）

- 静态变量/方法属于类，不属于对象
- 静态方法里不能直接用实例成员

---

## 5. 访问修饰符

- `public`：全世界可见
- `protected`：同包 + 子类
- 默认（不写）：同包
- `private`：仅本类

---

## 6. 方法重载

同名方法，参数不同，就是重载。

---

## 7. 常见内置方法

- `toString()`：对象打印
- `equals()`：比较内容
- `hashCode()`：哈希

## 9. 面试高频问答

### Q1：构造器能被继承吗？
**答：**
不能，但子类会调用父类构造器。

### Q2：static 能修饰构造器吗？
**答：**
不能，构造器必须依赖对象。

---

## 10. 一句话总结

类是模板，对象是实例，构造器 + this + static 是最核心三件套。
