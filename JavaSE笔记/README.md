# JavaSE 学习笔记 (基于《Java 核心技术 卷 I》)

此文件夹根据《Java 核心技术 卷 I》的章节逻辑进行组织，记录 Java 基础及核心特性的学习笔记。

## 目录结构
- `01-Java程序设计概述`: Java 特性、白皮书术语、Java 历史。
- `02-Java程序设计环境`: 安装 JDK、使用命令行工具、使用 IDE。
- `03-Java的基本程序设计结构`: 数据类型、变量、运算符、字符串、输入输出、控制流、大数值、数组。
- `04-对象与类`: 面向对象概述、预定义类、自定义类、静态字段与方法、方法参数、对象构造、包。
- `05-继承`: 类、子类、超类、多态、动态绑定、抽象类、Object 类、泛型数组列表、自动装箱、反射。
- `06-接口、Lambda表达式与内部类`: 接口特性、回调、Lambda 表达式、内部类、代理。
- `07-异常、断言与日志`: 处理错误、捕获异常、使用异常技巧、使用断言、日志。
- `08-泛型程序设计`: 泛型类、泛型方法、类型变量限定、泛型代码与虚拟机、继承规则、通配符。
- `09-集合`: 集合接口、具体集合（List, Set, Map, Queue）、迭代器、算法、遗留集合。
- `10-并发`: 线程、中断、线程状态、线程属性、同步、阻塞队列、并发集合、执行器（线程池）。
- `11-Swing与GUI`: (可选) Swing 基础、事件处理、用户界面组件。
