# Java 程序设计概述（大白话）

Java 就是“写一次，到处跑”的语言。你写代码，交给 JVM 去跑，JVM 负责在不同系统上翻译执行。

---

## 1. Java 的三件套

- **JDK**：开发工具箱（含编译器 + JRE）
- **JRE**：运行环境（含 JVM 和核心类库）
- **JVM**：真正跑字节码的“发动机”

一句话：开发用 JDK，运行靠 JVM。

---

## 2. Java 的特点（面试常问）

- 跨平台：靠 JVM
- 面向对象：一切皆对象
- 自动垃圾回收：不用手动释放内存
- 安全：有权限和沙箱机制

---

## 3. 程序怎么跑起来

```text
.java 源码 -> javac 编译 -> .class 字节码 -> JVM 解释/运行
```

---

## 4. 面试高频问答

### Q1：JDK/JRE/JVM 区别？
**答：**
JDK = JRE + 开发工具；JRE = JVM + 类库；JVM 是执行引擎。

### Q2：为什么 Java 能跨平台？
**答：**
编译成统一的字节码，靠不同平台的 JVM 去执行。

### Q3：Java 是解释型还是编译型？
**答：**
先编译成字节码，再由 JVM 解释/即时编译执行，属于混合型。

---

## 5. 一句话总结

Java 的核心是 JVM，理解 JDK/JRE/JVM 和编译流程就稳了。
