## 2. 面试高频问答

### Q1：受检异常和运行时异常区别？
**答：**
受检异常编译期强制处理；运行时异常可能运行期爆。

### Q2：finally 一定执行吗？
**答：**
基本一定，除非 JVM 直接退出。
