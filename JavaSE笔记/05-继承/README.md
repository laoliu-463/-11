## 2. Object 类必背

所有类默认继承 `Object`，常见方法：
- `toString()`：对象打印
- `equals()`：内容比较
- `hashCode()`：哈希规则

---

## 3. 面试高频问答

### Q1：为什么 Java 不支持多继承？
**答：**
为避免方法冲突（菱形问题），用接口解决。

### Q2：重写和重载的区别？
**答：**
重写是子类改父类方法；重载是同类方法参数不同。
