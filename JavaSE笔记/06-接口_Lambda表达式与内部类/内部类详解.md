# 内部类详解：把类写在类里

内部类就是“类中类”，常用来隐藏实现细节。

---

## 1. 常见内部类类型

- 成员内部类：类里直接定义
- 静态内部类：加 `static`
- 局部内部类：方法里定义
- 匿名内部类：一次性用完

---

## 2. 典型用法

匿名内部类：

```java
Runnable r = new Runnable() {
    public void run() { System.out.println("run"); }
};
```

Lambda 就是它的简化版（见 Lambda 笔记）。

---

## 3. 面试高频问答

### Q1：静态内部类能访问外部类实例吗？
**答：**
不能直接访问，只能用外部类对象去访问。

### Q2：匿名内部类的作用？
**答：**
快速实现接口或抽象类，写一次用一次。

---

## 4. 一句话总结

内部类是“就近封装”，匿名内部类是“一次性实现”。
